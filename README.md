# Groundwater Flow Deep Operator Emulator (PyTorch)

本项目旨在复现论文 [Taccari et al. (2024)](https://doi.org/10.1016/j.jhydrol.2023.130551) 中基于 DeepOnet/NO 的地下水流仿真器。

## 项目背景
原论文使用 **JAX** 框架实现了 2D 地下水单井抽水试验。本项目尝试将其迁移至 **PyTorch** 框架，以探索深度算子网络在水文建模中的通用性。

## 复现进展
* **2D 实验 (E1)**：尝试复现论文中的 2D 抽水试验模型，但目前未能达到预期精度，预测结果与基准数据存在显著偏差。
* **1D 简化测试**：为定位框架迁移中的问题，目前退回 1D 情况进行基准测试。

## 当前问题
初步测试显示 1D 模拟效果不理想：
* **拟合欠佳**：预测的降深曲线（Drawdown Cone）与参考解无法有效重合。
* **梯度捕捉**：模型难以准确刻画抽水井附近的剧烈水头变化。



## 后续排查
1.  对比 JAX 与 PyTorch 在算子初始化上的数值差异。
2.  优化 1D 场景下的超参数及数据归一化策略。
